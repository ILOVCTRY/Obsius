#### WhatWeb

kali 自带安装

普通扫描

```
whatweb 域名/文件（文件中存放多个域名）
```

详细扫描

```
whatweb -v 域名 
```

扫描强度

有四种强度 1-4

1. stealthy 每个目标发送一次 http 请求，并且会跟随重定向
2. unused 不可用（从 2011 年开始，此参数就是在开发状态）
3. aggressive 每个目标发送少量的 http 请求，这些请求时根据参数为 1 时结果确定的
4. heavy 每个目标会发送大量的 http 请求，会去尝试每一个插件

```
whatweb -v -a 等级 域名
```

内网扫描

```
whatweb --no-errors -t 255 IP/子网掩码
```

#### 常用参数

- `--alive=True`：只导出存活子域（默认False，导出全部）
- `--dns=True`：启用DNS解析（默认True）
- `--req=True`：启用HTTP请求验证（默认True）
- `--port=small`：指定端口探测范围（可选'default'、'small'、'large'）
- `--fmt=csv`：结果保存格式（支持csv、json）
- `--path=PATH`：自定义结果保存路径
- `--takeover=False`：检查子域接管（默认False）
- `--brute=True`：启用爆破模块
